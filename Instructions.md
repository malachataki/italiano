# Οδηγίες ανεβάσματος στο GitHub Pages

## Βήμα 1 — Δημιούργησε νέο repository στο GitHub

1. Πήγαινε στο https://github.com/new
2. Repository name: `italiano` (ή ό,τι θέλεις)
3. Βάλε το **Public** ✓
4. Κλικ **Create repository**

## Βήμα 2 — Ανέβασε τα αρχεία

Έχεις δύο επιλογές:

### Α) Μέσω browser (πιο εύκολο)
1. Στη σελίδα του νέου repository, κλικ **uploading an existing file**
2. Σύρε ΟΛΑ τα αρχεία του φακέλου:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
3. Κλικ **Commit changes**

### Β) Μέσω Git (αν έχεις εγκατεστημένο)
```
git clone https://github.com/ΧΡΗΣΤΗΣ_ΣΟΥ/italiano.git
# Αντέγραψε τα αρχεία μέσα
git add .
git commit -m "first commit"
git push
```

## Βήμα 3 — Ενεργοποίησε το GitHub Pages

1. Settings (⚙️) → **Pages** (αριστερά)
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Κλικ **Save**
5. Περίμενε 1-2 λεπτά

## Βήμα 4 — Η εφαρμογή σου είναι live!

Διεύθυνση: `https://ΧΡΗΣΤΗΣ_ΣΟΥ.github.io/italiano/`

## Εγκατάσταση στο κινητό (Android)

1. Άνοιξε τη διεύθυνση στο Chrome
2. Μενού (⋮) → **Προσθήκη στην αρχική οθόνη**
3. Κλικ **Προσθήκη** → εμφανίζεται σαν κανονική εφαρμογή!

## Εγκατάσταση στο κινητό (iPhone/iPad)

1. Άνοιξε τη διεύθυνση στο Safari
2. Κουμπί κοινής χρήσης (□↑) → **Προσθήκη στην οθόνη Αφετηρίας**
3. Κλικ **Προσθήκη**

## Εγκατάσταση στον υπολογιστή (Chrome/Edge)

1. Άνοιξε τη διεύθυνση
2. Στη μπάρα διευθύνσεων θα δεις εικονίδιο εγκατάστασης (⊕)
3. Κλικ **Εγκατάσταση** → τρέχει σαν ξεχωριστό παράθυρο!
